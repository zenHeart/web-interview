## 是否细心

### 1. 末尾分号

```js
const length = 4;
const numbers = [];
for (var i = 0; i < length; i++);
{
    numbers.push(i + 1);
}
numbers; // => ???
```

**注意末尾的分号会导致执行结果输出为 [5] 而非 [1,2,3,4]**

> 该题着重考察应聘者是否足够细心
