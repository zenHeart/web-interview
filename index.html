前端面试

项目概述

项目汇总了 github 及网络上大量的前端面试题。 除了简单的汇总,笔者做了如下工作。

  1. 分类和汇总,并不是简单的问题罗列,而是对所有问题进行了分类,并将多个类似或考点相近的问题合并为了单一问题,方便更有针对性的复习?
  2. 延伸提高, 提供完善的答案和延伸阅读材料,虽然面试是短期准备,但是功在平时

面试资料分为如下几部分

  1. 计算机基础 放置通用的计算机知识
  2. 前端基础 放置前端相关技术栈的基本领域知识,重点考察基本技能
  3. 前端进阶 在掌握基础知识的前提下,重点考察项目经验和前端开发过程中的工程经验,设计开发流程中各环节即延伸知识,不局限于前端领域
  4. 面试技巧 包含软技能的知识点,例如开发习惯,编程品质,面试技巧等方面的知识。
  5. 公司题库 收集网络中相关公司的试题,帮助面试者知晓面试流程

试题等级

按照面试题的出现频率和重要性,给出如下等级。 若准备时间不足则只需看等级较高的试题即可。

星级 含义
⭐️⭐️⭐️⭐️⭐️ 必须掌握
⭐️⭐️⭐️⭐️ 重要
⭐️⭐️⭐️ 较重要
⭐️⭐️ 有概念
⭐️ 知道即可

对于开发经验不同的面试者,侧重点会有所区别,例如校招则必须掌握 5 星内容 而对于有多年开发经验的人员除了必须掌握 3-5 星内容外, 1-2 星才是重点
