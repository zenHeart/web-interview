## http1、2、3区别 {#p1-http-1-2-3-difference}

1. HTTP/1.1
   * 持久连接（Keep-Alive）
   * 管道化请求
   * 存在队头阻塞问题
2. HTTP/2
   * 多路复用
   * 二进制分帧
   * 服务器推送
   * 头部压缩（HPACK）
3. HTTP/3
   * 基于 QUIC 协议（UDP）
   * 0-RTT 快速握手
   * 避免队头阻塞
   * 内置加密（TLS 1.3）
