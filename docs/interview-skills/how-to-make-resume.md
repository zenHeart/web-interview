# 简历命名
* `应聘岗位+姓名+电话` 的方式
* 采用 pdf
* 倒 L 型，从左至右，从上到下。
* 超强技能点描述举例

```
读了 APUE，实现了一个简单的 Shell，支持重定向和管道。代码在 xxx
```
