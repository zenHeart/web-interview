**null undefined 区别?**

1. 语言层面
   1. **null** 表示值为空
   2. **undefined** 表示缺省值或默认值
2. 功能层面:
   1. 数值转换
      1. null 会变为 0 
      2. undefined 变为 NaN

详细区别参考 [阮一峰 undefined 与 null 的区别](https://www.ruanyifeng.com/blog/2014/03/undefined-vs-null.html)
